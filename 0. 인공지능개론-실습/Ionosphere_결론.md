# 이오노스피어 데이터셋 의사결정 트리 분류 결론

의사결정 트리 모델은 이오노스피어 데이터셋에서 88.57%의 높은 정확도를 달성하였으며, ROC 곡선의 AUC 값이 0.889로 나타나 모델이 두 클래스를 효과적으로 구분하는 우수한 성능을 보여주었습니다. 대부분의 예측 점수가 0 또는 1에 가까운 확실한 값으로 나타나 모델이 높은 신뢰도로 분류를 수행하고 있음을 확인할 수 있습니다. 

## 코드 설명

본 코드는 이오노스피어(Ionosphere) 데이터셋을 활용하여 이진 분류 문제를 해결하는 전체 과정을 MATLAB으로 구현한 예제입니다. 주요 단계는 다음과 같습니다.

1. **데이터 불러오기 및 전처리**  
   - `readtable` 함수를 이용해 CSV 파일로부터 데이터를 불러오고, 입력 변수(X)와 클래스 레이블(class)을 분리합니다.
   - 클래스 레이블을 범주형(categorical)으로 변환한 뒤, `grp2idx`를 사용해 0과 1로 이루어진 이진 레이블(y)로 변환합니다.

2. **데이터 분할**  
   - `cvpartition`을 사용해 전체 데이터를 70% 훈련, 30% 테스트 세트로 무작위 분할합니다.

3. **의사결정 트리 모델 학습**  
   - 훈련 데이터(XTrain, yTrain)를 이용해 `fitctree`로 의사결정 트리 분류기를 학습합니다.

4. **모델 예측 및 성능 평가**  
   - 테스트 데이터(XTest)에 대해 예측을 수행하고, 예측 결과와 실제 레이블을 비교하여 정확도(accuracy)를 계산합니다.
   - 각 샘플이 각 클래스에 속할 확률(score)도 함께 출력합니다.

5. **트리 시각화**  
   - `view(treeModel, 'Mode', 'graph')`를 통해 학습된 트리 구조를 시각적으로 확인할 수 있습니다.

6. **ROC 곡선 및 AUC 계산**  
   - `perfcurve` 함수를 사용해 ROC 곡선과 AUC(곡선 아래 면적)를 계산하고, 이를 그래프로 시각화합니다.
   - AUC 값은 모델의 분류 성능을 종합적으로 평가하는 지표입니다.

7. **테스트 데이터 샘플별 점수 및 순위화**  
   - 테스트 데이터의 각 샘플에 대해 예측 점수를 기반으로 내림차순 정렬하여, 신뢰도가 높은 순서대로 결과를 테이블로 출력합니다.

---

## 결론

이 MATLAB 코드는 이오노스피어 데이터셋에 대해 의사결정 트리 분류기를 적용하는 전 과정을 체계적으로 다루고 있습니다. 데이터 전처리부터 모델 학습, 성능 평가(정확도, ROC, AUC), 그리고 개별 샘플의 신뢰도 기반 순위화까지 실습할 수 있습니다.  
실험 결과, 모델은 약 88.57%의 정확도와 0.889의 AUC를 기록하여, 두 클래스를 효과적으로 구분하는 우수한 분류 성능을 보였습니다. 또한, ROC 곡선과 샘플별 점수 분석을 통해 모델의 신뢰도와 분류 경향을 시각적으로 확인할 수 있습니다.  
이 코드는 머신러닝 분류 문제의 전형적인 워크플로우를 이해하고, MATLAB 환경에서 실습하는 데 매우 유용한 예제입니다.

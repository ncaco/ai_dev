1-1. 단일 예제 (x, y)에 대한 제곱 손실(squared loss)

제곱 손실은 "예측한 값"과 "정답(실제 값)"의 차이를 제곱해서 계산합니다.

예를 들어,
- 우리가 예측한 값이 h_w(x) = w_1x + w_0 이고,
- 실제 정답이 y 라고 하면,

제곱 손실은 이렇게 계산합니다:
L(w) = (예측값 - 실제값)² = (w_1x + w_0 - y)²

즉, 예측이 정답과 멀수록(차이가 클수록) 손실 값이 커집니다.

1-2. w_0와 w_1의 업데이트 규칙

경사 하강법에서는 손실 함수의 기울기(경사)를 계산하여 그 반대 방향으로 파라미터를 업데이트합니다.

먼저 손실 함수의 각 파라미터에 대한 편미분을 계산합니다:

∂L/∂w_0 = 2(w_1x + w_0 - y) · 1 = 2(w_1x + w_0 - y)
∂L/∂w_1 = 2(w_1x + w_0 - y) · x = 2(w_1x + w_0 - y) · x

경사 하강 업데이트 규칙은 다음과 같습니다(η는 학습률):

w_0 := w_0 - η · ∂L/∂w_0 = w_0 - η · 2(w_1x + w_0 - y)
w_1 := w_1 - η · ∂L/∂w_1 = w_1 - η · 2(w_1x + w_0 - y) · x

또는 간단히:

w_0 := w_0 - 2η(w_1x + w_0 - y)
w_1 := w_1 - 2η(w_1x + w_0 - y) · x

여러 데이터 포인트에 대해서는 각 포인트에서의 기울기의 합(또는 평균)을 사용합니다.
